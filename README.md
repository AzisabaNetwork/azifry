# アジ鯖Docker
🐋

## 📕 使い方
### 起動とリロード
```sh
docker-compose build --pull
docker-compose pull
docker-compose up -d --remove-orphans
docker system prune -f --volumes
```
### 再起動
```sh
docker-compose up -d --remove-orphans --force-recreate
```
### シャットダウン
```sh
docker-compose down --remove-orphans
```
